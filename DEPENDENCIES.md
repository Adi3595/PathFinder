# Project Dependencies

To set up this project locally, you will need to install the following dependencies.

## Core Dependencies

```bash
npm install @google/genai clsx d3 dotenv express firebase framer-motion lucide-react motion react react-dom react-markdown react-router-dom recharts tailwind-merge vite
```

## Development Dependencies

```bash
npm install -D @firebase/eslint-plugin-security-rules @types/express @types/node @types/react @types/react-dom @vitejs/plugin-react autoprefixer tailwindcss tsx typescript
```

## Installation Instructions

1. Clone the repository
2. Run `npm install` to install all dependencies listed in `package.json`
3. Set up your `.env` file based on `.env.example`
4. Run `npm run dev` to start the development server
